# Kiro Workshop

Hands-on labs to explore Kiro's agentic and context engineering capabilities.

## Getting Started

```bash
git clone https://github.com/awsgcr/kiro-workshop
cd kiro-workshop
```

Launch a lab with Kiro:

```bash
kiro lab01-capture-learning-into-kiro-power
```
